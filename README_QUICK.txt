================================================================================
  БЫСТРЫЙ СТАРТ (3 КОМАНДЫ)

Вы хотите просто начать работать? Вот ВСЁ что нужно:

================================================================================
КОМАНДА 1: УСТАНОВКА (30 секунд)
================================================================================

pip install -r requirements.txt

================================================================================
КОМАНДА 2: ОБУЧЕНИЕ (2-5 минут) ⚠️ ОБЯЗАТЕЛЬНО!
================================================================================

python3 train.py

Система изучит ВСЕ ваши товары из product1.xlsx и sku_vkus.xlsx:
  ✓ 2035 брендов
  ✓ 3437 вкусов  
  ✓ Все категории
  ✓ Паттерны в названиях

Это делается ОДИН РАЗ!

================================================================================
КОМАНДА 3: ИСПОЛЬЗОВАНИЕ
================================================================================

СПОСОБ А (интерактивный):
-------------------------
python3 start.py

Выберите действие из меню.


СПОСОБ Б (автоматический):
--------------------------
python3 process_folder.py

Укажите папку с файлами, программа сделает всё сама.


СПОСОБ В (командная строка):
----------------------------
python3 auto_categorizer.py -i new_products.csv

================================================================================
ГОТОВО!
================================================================================

Теперь система знает ВСЕ ваши бренды и вкусы!

Файлы с результатами:
  - product1_updated.xlsx
  - sku_vkus_updated.xlsx

================================================================================
ВАЖНО!
================================================================================

Если вы забыли запустить обучение (команда 2), при попытке обработки
товаров вы увидите сообщение:

  ⚠️ База знаний не найдена!
  Сначала запустите обучение: python3 train.py

Просто запустите train.py и всё заработает!
