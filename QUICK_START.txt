📋 ЧТО ЭТО?
───────────
Интерактивный BI дашборд для анализа продаж с графиками план/факт,
метриками, анализом по сетям, продуктам и финансовыми показателями.

🎯 БЫСТРЫЙ ЗАПУСК (3 ШАГА)
───────────────────────────

1️⃣ УСТАНОВИТЕ PYTHON
   Скачайте с https://www.python.org/downloads/
   ⚠️ При установке ОБЯЗАТЕЛЬНО поставьте галочку "Add Python to PATH"!

2️⃣ ЗАПУСТИТЕ ДАШБОРД
   
   Способ А (Самый простой - для Windows):
   ────────────────────────────────────────
   • Дважды кликните на файл: run_dashboard.bat
   • Дождитесь установки библиотек (первый раз займет 1-2 минуты)
   • Откроется дашборд
   
   Способ Б (Для всех ОС):
   ───────────────────────
   • Откройте командную строку (cmd или terminal)
   • Перейдите в папку со скриптами:
     cd "путь\к\папке\со\скриптами"
   • Запустите:
     python run_dashboard.py

3️⃣ ОТКРОЙТЕ В БРАУЗЕРЕ
   Перейдите по адресу: http://localhost:8050

📂 НАСТРОЙКА ПУТИ К ФАЙЛУ data.xlsx
────────────────────────────────────

Ваш файл находится здесь:
\\FS\Users\Private\GFD\Public\Трейд-маркетинг\7.Общие документы\Гусев\итог\data.xlsx

Вариант 1 (Рекомендуется):
──────────────────────────
Скопируйте все файлы (.py и .bat) в папку с data.xlsx:
\\FS\Users\Private\GFD\Public\Трейд-маркетинг\7.Общие документы\Гусев\итог\

Затем просто запустите run_dashboard.bat

Вариант 2:
─────────
Оставьте скрипты где угодно, путь уже прописан в run_dashboard.bat

Вариант 3:
─────────
Измените путь в файле run_dashboard.bat:
Откройте его в блокноте и измените строку:
set DATA_FILE_PATH=ваш\новый\путь\data.xlsx

📊 ЧТО ПОКАЗЫВАЕТ ДАШБОРД?
──────────────────────────

✅ KPI Метрики:
   • Продажи (руб и шт) - план/факт/выполнение
   • Доход - план/факт/выполнение
   • Затраты - план/факт

✅ Графики:
   • План/Факт продаж по месяцам
   • Динамика выполнения плана
   • Доходы, затраты и рентабельность
   • Разбивка затрат по типам
   • ТОП-15 торговых сетей
   • ТОП-15 продуктов
   • Распределение по группам сбыта

✅ Интерактивность:
   • Наведите курсор для детальной информации
   • Приближайте графики мышью
   • Скрывайте/показывайте данные кликом на легенду
   • Сохраняйте графики как изображения

❓ РЕШЕНИЕ ПРОБЛЕМ
──────────────────

Проблема: "Python не найден"
Решение: Установите Python и добавьте в PATH при установке

Проблема: "Файл data.xlsx не найден"
Решение: Проверьте путь в run_dashboard.bat или скопируйте data.xlsx
        в папку со скриптами

Проблема: "Порт 8050 занят"
Решение: Закройте другие программы на порту 8050 или измените порт
        в файле bi_dashboard.py (последняя строка)

Проблема: Дашборд не открывается
Решение: Откройте браузер вручную и введите: http://localhost:8050
